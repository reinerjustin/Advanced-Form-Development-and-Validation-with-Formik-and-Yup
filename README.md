# Advanced-Form-Development-and-Validation-with-Formik-and-Yup

The Expo project is located in the `Ass3` folder.

## Getting Started

1. Clone the repository.
2. Navigate to the project folder:

   cd "Ass3"

3. Install dependencies:

   npm install

4. Run the project:

   npx expo start
