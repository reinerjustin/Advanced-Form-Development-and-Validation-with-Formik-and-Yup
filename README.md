## Assignment Overview 

This assignment requires students to design and implement professional-quality forms in a mobile application using React Native with Expo, Formik for form state management, and Yup for schema-based validation. Working in groups of three, students will create multiple real-world forms and demonstrate strong validation logic, user experience design, and collaborative development practices. 

## Learning Outcomes 

Upon successful completion of this assignment, students will be able to: 
• Implement complex forms using Formik in an Expo environment 
• Apply advanced schema-based validation using Yup 
• Design professional and user-friendly form interfaces 
• Implement authentication-style forms following industry standards 
• Collaborate effectively using GitHub version control
